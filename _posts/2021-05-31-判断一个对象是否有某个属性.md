### 判断一个对象是否具有某个属性，以前只知道 hasOwnProperty 这个方法。最近看到还有 in 这种方式，故记录一下。

```
let a = {
    b: 1
}
```

1. hasOwnProperty

```
a.hasOwnProperty('b') // true
```

2. in

```
('b' in a) // true
```
